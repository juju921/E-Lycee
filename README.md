E-Lycee
=======

#Pré-requis

- MySQL 5.6
- Php 5

#Installation

1. Cloner
2. Exécuter : 
  - "php composer.phar install"
    - database name : elycee
    - pour le reste, laisser par defaut
  - "php app/console assets:install --symlink"
  - "php app/console doctrine:database:create"
  - "php app/console doctrine:schema:update --force"
